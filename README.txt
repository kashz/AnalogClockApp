- 概要
OpenGL，GLUTを使って作られたグラフィカルなアナログ時計です．
デフォルトの時計とシンプルな時計の2種類の時計と
ノーマルモードとナイトモードの2種類の配色で計4種類の時計を見ることができます．

- Build方法

Buildのためには以下が必要です．
Mac
    Xcodeもしくは，Developer Tools．
Windows
    フルインストールされたCygwin

以下のコマンドでビルドできます．
Windowsの場合はCygwinで，Macの場合は標準インストールされている端末から実行してください．
$はプロンプト記号を示しているため，それ以降を入力してください．
Windowsの場合はアイコンが付きます．
-----------------
$ make
-----------------


- 実行方法

上記コマンドによって生成されたj13411.exeを実行します．
-----------------
$ ./j13411
-----------------
以下のオプションをつけることでシンプルな時計で実行されます．
-----------------
$ ./j13411 --simple
-----------------
以下のオプションをつけることでナイトモードで実行されます．
-----------------
$ ./j13411 --night
-----------------


- 使い方

実行後も時計の種類とモード(配色)を変更することができます．
キーボードで"t"を入力すると時計の種類が変わります．
キーボードで"m"を入力するとモードが切り替わります．
"q"かエスケープキーで終了します．

- ディレクトリ構成

ディレクトリ構成は以下のようになっています．
------------------
├── Makefile
├── README.md
├── include       // Headers
├── media         // Resources (e.g. images)
├── icon.rc       // Resource file to link icon
├── obj           // Object files
└── src           // C files
------------------
Cファイルはsrcディレクトリに，ヘッダーファイルはincludeディレクトリに入っています．
icon.rcはWindowsのみでアイコンをリンクするために使われます．
